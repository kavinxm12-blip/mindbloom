# MindBloom 🌱

Track your emotions. Understand yourself. Grow every day.

A modern, production-ready Emotional Health Journal web application built with Next.js, React, TypeScript, and Supabase.

## 🎯 Features

- **Emotion Tracking** - Track 10+ emotions with unique animations and gradients
- **Smart Journal** - Rich text editor with markdown support
- **AI Insights** - Emotion analysis and personalized suggestions
- **Beautiful Analytics** - Mood heatmaps and trends
- **Dynamic Themes** - Automatic theme switching
- **Goal & Habit Tracking** - Build healthy habits
- **Premium UI** - Glassmorphism and smooth interactions

## 🛠 Tech Stack

- **Frontend**: Next.js 15, React 19, TypeScript, Tailwind CSS
- **Backend**: Supabase, PostgreSQL
- **Animations**: Framer Motion
- **Charts**: Recharts
- **Deployment**: Vercel

## 📦 Quick Start

### Prerequisites
- Node.js 18+
- npm or yarn
- Supabase account

### Installation

```bash
git clone https://github.com/kavinxm12-blip/mindbloom.git
cd mindbloom
npm install
```

### Setup Environment

```bash
cp .env.example .env.local
```

Update `.env.local` with your Supabase credentials.

### Run Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## 📁 Project Structure

```
mindbloom/
├── app/              # Next.js app router
├── components/       # React components
├── features/         # Feature modules
├── lib/              # Utilities
├── types/            # TypeScript types
├── constants/        # App constants
├── styles/           # Global styles
```

## 🚀 Deployment

Deploy on Vercel:

```bash
npm run build
npm start
```
